# get_code API

Minimal HTTP API service scaffold with a single `/api/ping` endpoint and a browser-based update button.

## Requirements
- Python 3.8.x (Win7 and Windows Server 2012 R2 compatible)
- Git (for the update button)

## Quick Start (Windows)
```bat
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python src\server.py
```

## Quick Start (macOS/Linux)
```bash
python -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
python src/server.py
```
